ABOUT CineSync

CineSync uses advanced machine learning to analyze your unique taste profile and instantly curate hidden gems.  
No complex surveys, ratings, or login required—just type a movie you loved and get a personalized watchlist in seconds.  
CineSync brings the joy back to discovering films, from blockbusters to hidden classics.  

FEATURES

- Input Your Favorite Movie: Type a movie you loved, like Avatar or Interstellar.  
- Instant Results: Receive AI-curated recommendations immediately.  
- No Login Required: Privacy first, always.  
- Personalized Match Scores: See how well each movie matches your taste.  
- Seamless Design: Works perfectly on desktop and mobile.  
- Curated Watchlist: Hidden gems, classics, and trending favorites.  

HOW IT WORKS

1. Type a Movie You Loved: Examples include AVATAR, HOME, MARUTO.  
2. AI Analysis: Our algorithm evaluates thousands of data points including genre, director, plot, and mood.  
3. Curated Recommendations: Get a watchlist of movies you’ll actually enjoy.  
4. Instant Watchlist: Ranked by match percentage, ready for your next binge.  

TECH STACK

CineSync is built with modern, high-performance tools:  
- Frontend: Tailwind CSS, HTML, JavaScript, Iconify  
- Backend: Python, Flask, Replit Hosting  
- Data & AI: Pandas, NumPy, Vector Similarity Models, Optional NLP Embeddings
